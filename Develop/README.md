# E-Commerce Module 13
Databases of categories, products, and tags.

## How to run to terminal

- open terminal
- mysql -u root
- source db/schema.sql
- quit mysql
- npm run seed
- npm start

## Technologies used

- javascript
- npm packages
- mysql
- apis
